# Simple Shell

A simple UNIX command interpreter written as part of the Holberton School curriculum.

## Description

This is a simple UNIX command line interpreter (shell) that replicates basic functionality of the shell. It can execute commands, handle built-ins, and manage processes.

## Features

- Displays a prompt and waits for user input
- Executes commands with their arguments
- Handles the PATH environment variable
- Implements built-in commands like `exit` and `env`
- Supports command line arguments
- Handles errors and edge cases
